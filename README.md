# Quantix - CIT-U Laboratory Inventory Management System

## Overview

Quantix is a centralized inventory management system designed for CIT-U laboratories. It aims to support NAS and laboratory coordinators in managing, monitoring, and maintaining laboratory equipment and supplies.

## Objectives

- Develop a secure and centralized inventory management system for CIT-U laboratories
- Implement Microsoft OAuth authentication limited to institutional email accounts
- Apply a three-tier system architecture using Spring Boot, React, and Android
- Provide a consistent and responsive user experience across web and mobile platforms
- Enable seamless communication for backend, web application, and mobile application through RESTful APIs

## Features

- **Microsoft OAuth login and register** using CIT-U institutional email
- **User Profile Management** - View and update personal profile information
- **Role-Based Access Control** for NAS and laboratory coordinators
- **Transaction History** - Complete audit trail of all actions
- **Asset Management** - Categorization, condition, and status tracking
- **Inventory Management** - Creation, update, and archival of equipment
